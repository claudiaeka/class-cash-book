# Clash Cash Book 📊

A Python-based cash book management system for recording transactions, calculating balances, and generating financial reports.

## Features ✨

- **Record Transactions**: Easily log income (credits) and expenses (debits)
- **Balance Management**: Automatic balance calculation and tracking
- **Financial Reports**: Generate detailed transaction ledgers with summaries
- **Date Range Filtering**: View transactions for specific periods
- **Data Persistence**: Save and load data to/from JSON files
- **Transaction History**: Complete audit trail with transaction IDs
- **Summary Statistics**: Quick overview of financial status
- **Error Handling**: Validation for insufficient funds and invalid inputs
